# Оценка 3 — Матрица и определитель

## Условие

Пользователь задаёт N (1–3). Создать матрицу NxN, заполнить случайными числами, вычислить определитель.

## Формулы определителя

| Размер | Формула |
|---|---|
| 1x1 | `det = a[0][0]` |
| 2x2 | `det = a00*a11 - a01*a10` |
| 3x3 | Правило Саррюса |

### Правило Саррюса (3x3)

```
det = a00*a11*a22 + a01*a12*a20 + a02*a10*a21
    - a02*a11*a20 - a00*a12*a21 - a01*a10*a22
```

## Пример вывода

```
Введите N (1-3): 2

Матрица 2x2:
   3  -1
   2   5

Определитель: 17
```

## Сборка и запуск

```bash
gcc zadanie1.c -o grade3
./grade3
```
