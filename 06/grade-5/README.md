# Оценка 5 — Метод Крамера (произвольное N)

## Условие

Реализовать алгоритм Крамера для решения СЛАУ любого размера N (в том числе N=8).

## Особенности реализации

- Динамическое выделение памяти через `malloc`
- Определитель вычисляется методом Гаусса с выбором ведущего элемента
- Освобождение памяти после работы

## Используемые функции

| Функция | Описание |
|---|---|
| `alloc_matrix(n)` | Выделяет память под матрицу NxN |
| `free_matrix(a, n)` | Освобождает память матрицы |
| `copy_matrix(dst, src, n)` | Копирует матрицу |
| `det_gauss(A, n)` | Вычисляет определитель методом Гаусса |
| `replace_col(T, A, b, n, col)` | Заменяет столбец на вектор b |
| `print_matrix(A, b, n)` | Выводит расширенную матрицу (A│b) |
