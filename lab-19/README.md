# Lab 19 — Консольный банкомат (TUI)

C++17, ANSI TUI, навигация стрелками, бинарный журнал, recovery.

## ⚡ Быстрый запуск (одна команда)

```bash
bash lab-19/run.sh
```

Или из папки `lab-19/`:

```bash
bash run.sh
```

Скрипт сам собирает проект (`cmake + make`) и запускает бинарь.

> **Требования:** `cmake ≥ 3.20`, `g++ / clang++ с C++17`, Linux/macOS.

