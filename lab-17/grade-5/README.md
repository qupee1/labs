# Оценка 5 — Полное покрытие + `ctest` + форматированный вывод

## Отличия от grade-4

- **Каждый кейс** — отдельная тест-функция (а не таблица в одной), чтобы каждый получил свой `[ RUN ] ... [ OK ]` и свою строку в выводе `ctest`.
- Сборка через **корневой CMakeLists.txt** в [lab-17/](..), регистрирует тест в `ctest` как target.
- Вывод как в примере из методички: `100% tests passed, 0 tests failed out of N`.

## Что покрывается

23 тест-функции, разбитые на 7 групп:

| Группа | Кейсы |
|---|---|
| Арифметика | `+`, `-`, `*`, `/` |
| Приоритет / скобки | `2+3*4`, `(2+3)*4`, унарный минус |
| Степень | `2^10`, `2^3^2` ← левая ассоциативность (фиксируем фактическое поведение) |
| Встроенные функции | `sqrt`, `abs`, `pow`, `ln`, константа `pi` |
| Тригонометрия | `sin(0)`, `cos(0)`, `sin² + cos² ≡ 1` |
| Особые значения | `1/0 → inf`, `sqrt(-1) → NaN` |
| Ошибки парсера | `1 +`, `(1+2`, `foo(1)`, пустая строка |

## Запуск через ctest

Сборка идёт из **корня lab-17**, не из этой папки:

```bash
cd ..               # в lab-17/
cmake -B build
cmake --build build
cd build
ctest --output-on-failure
```

Ожидаемый вывод:

```
Test project .../lab-17/build
    Start 1: grade-3_simple
1/3 Test #1: grade-3_simple ...................   Passed    0.00 sec
```

То же, что в примере на скриншоте PDF.

## Полезные ключи `ctest`

| Ключ | Что делает |
|---|---|
| `--output-on-failure` | показывать stdout только упавших тестов |
| `-V` (`--verbose`) | весь вывод всех тестов |
| `-R <regex>` | запустить только тесты, имя которых matches regex |
| `-j N` | параллельный запуск в N потоков |
| `--rerun-failed` | прогнать только упавшие в прошлый раз |

## Найденная особенность tinyexpr

В ходе тестирования обнаружено, что `^` в tinyexpr — **лево-ассоциативный**:

```
2^3^2 = (2^3)^2 = 8² = 64
```

Это нестандартно — в математике и большинстве языков (Python, Haskell) `**` право-ассоциативный (`2^3^2 = 2^(3^2) = 512`). Тест `test_power_left_assoc` **фиксирует фактическое поведение** библиотеки. Если когда-нибудь tinyexpr перейдёт на правую ассоциативность — этот тест сразу упадёт и просигнализирует о breaking change.
