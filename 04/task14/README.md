# Задание 14 — Количество единичных битов

## Условие

Найти количество единичных битов в целом неотрицательном числе `a`.

## Метод решения

Используется **алгоритм Брайана Кернигана**: операция `a &= (a - 1)` сбрасывает (обнуляет) самый младший единичный бит числа. Повторяя эту операцию до тех пор, пока число не станет равно нулю, и считая количество итераций, получаем количество единичных битов.

Почему `a & (a - 1)` сбрасывает младший единичный бит:
- `a = 1100` (двоичное)
- `a - 1 = 1011`
- `a & (a-1) = 1000` — младший единичный бит сброшен

### Пример по шагам для числа 13 (`1101` в двоичном):

| Шаг | a (двоичное) | a-1 (двоичное) | a & (a-1) | Счётчик |
|-----|-------------|----------------|-----------|--------|
| 1 | `1101` | `1100` | `1100` | 1 |
| 2 | `1100` | `1011` | `1000` | 2 |
| 3 | `1000` | `0111` | `0000` | 3 |

Результат: 3 единичных бита.

## Используемые битовые операции

- `&=` — побитовое И с присваиванием

## Пример работы

```
Введите неотрицательное целое число a: 13
Количество единичных битов в числе 13: 3

Введите неотрицательное целое число a: 255
Количество единичных битов в числе 255: 8
```
