# RGR-5 — Шифр Полибия (Polibi / DePolibi)

## Описание

Реализация шифра Полибия на языке C:
- **Polibi** — шифрование открытого текста
- **DePolibi** — дешифровка шифртекста

Программа поддерживает два режима:
| Режим | Таблица | Алфавит |
|-------|---------|----------|
| Латиница | 5×5 | A–Z (I=J) |
| Кириллица | 6×6 | А–Я (без Ё, без Ъ) + пробел |

Язык определяется автоматически по наличию кириллических символов.

## Принцип работы

Каждая буква открытого текста заменяется парой цифр **RC** (строка и столбец в таблице).

Пример (латиница, ключ = стандартный алфавит):
```
Таблица 5×5:
  1 2 3 4 5
1 A B C D E
2 F G H I K
3 L M N O P
4 Q R S T U
5 V W X Y Z

"HELLO" → 24 15 33 33 34
```

## Критерии

- ✅ Проверка совпадения исходного текста с дешифрованным
- ✅ Нет динамической памяти (`malloc`/`free` не используются)
- ✅ Функции шифрования/дешифровки вынесены в статическую библиотеку `libcipher.a`

## Сборка и запуск

```bash
make          # сборка: libcipher.a + исполняемый файл polibi
./polibi      # запуск
make clean    # очистка
```
