# Оценка 5 — Вариант 1: Массивы X и Y

## Условие

Записать каждый второй элемент целочисленного массива X=(x1,x2,…,xn) подряд в массив Y=(y1,y2,…,yk). Определить количество простых чисел в каждом массиве. Вычислить среднее арифметическое всех элементов массивов X и Y.

## Используемые функции

| Функция | Описание |
|---|---|
| `printArray(arr, size)` | Вывод массива на экран |
| `isPrime(num)` | Проверка числа на простоту |
| `countPrimes(arr, size)` | Подсчёт простых чисел в массиве |
| `average(arr, size)` | Вычисление среднего арифметического |

## Алгоритм `isPrime`

Проверка числа на простоту за O(√n):
- Числа < 2 — не простые
- Число 2 — простое
- Чётные числа > 2 — не простые
- Делители проверяются от 3 до √num с шагом 2

## Пример вывода

```
Массив X: [ 3 8 7 10 5 6 2 9 13 4 ]
Массив Y (каждый второй элемент X): [ 8 10 6 9 4 ]

Количество простых чисел в X: 5
Количество простых чисел в Y: 0

Среднее арифметическое X: 6.70
Среднее арифметическое Y: 7.40
```
