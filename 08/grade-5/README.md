# Вариант 1 — Оценка 5

## Задание

То же, что и для оценки 4, но с использованием динамического выделения памяти (`malloc`/`free`) и вынесением подзадач в отдельные функции.

## Исправления относительно оригинала

Удалены бессмысленные/незаконченные строки `p2 = p1`, `*p2 =`, `**p2 = 5` из `main`, которые приводили к ошибке компиляции.

## Функции

- `allocate_matrix` — динамическое выделение двумерного массива
- `free_matrix` — освобождение памяти
- `enter_matrix` — ввод элементов
- `print_matrix` — вывод матрицы
- `find_max_position` — поиск максимального элемента
- `create_minor_matrix` — формирование минора
- `count_str_with_zeros` — подсчёт строк с нулями
- `print_minor_diagonal_reversed` — главная диагональ в обратном порядке

## Запуск

```bash
gcc zadanie1.c -o zadanie1
./zadanie1
```
