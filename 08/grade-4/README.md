# Вариант 1 — Оценка 4

## Задание

Сформировать матрицу B путём вычеркивания строки и столбца матрицы A, на пересечении которых находится максимальный элемент. Определить количество строк, в которых есть нули. Записать в обратном порядке главную диагональ матрицы B.

## Решение

Используются статические массивы `MAXN x MAXN`. Программа находит позицию максимального элемента, вычеркивает его строку и столбец, затем считает строки с нулями в полученной матрице B и выводит главную диагональ в обратном порядке.

## Запуск

```bash
gcc zadanie1.c -o zadanie1
./zadanie1
```

## Пример ввода/вывода

```
Ввод:
3 3
1 2 3
4 9 5
6 7 8

Вывод (матрица B):
1 3
6 8
0       <- строк без нулей
8 1     <- диагональ в обратном порядке
```
