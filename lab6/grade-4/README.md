# Оценка 4 — Метод Крамера (СЛАУ 2x2 и 3x3)

## Условие

Реализовать алгоритм Крамера для решения СЛАУ NxN, где N = 2 или 3.

## Метод Крамера

Для каждой переменной x_k:

```
x_k = det(A_k) / det(A)
```

где `A_k` — матрица A, в которой k-й столбец заменён вектором правых частей b.

## Проверочные тесты

| Тест | Система | Ответ |
|---|---|---|
| 1 | 5x+2y=7, 2x+y=9 | x=-11, y=31 |
| 2 | x-y=5, 2x+y=1 | x=2, y=-3 |
| 3 | 3x+2y=0.5, 2x+5y=4 | x=-0.5, y=1 |
| 4 | x1+2x2+3x3=5, 4x1+5x2+6x3=8, 7x1+8x2=2 | x=-2, y=2, z=1 |
| 5 | 2x-3y+z=-7, x+2y-3z=14, -x-y+5z=-18 | x=1, y=2, z=-3 |
| 6 | 3x+2y+z=1, 6x+5y+4z=-2, 9x+8y+7z=3 | Нет единственного решения |

## Пример вывода

```
Введите N (2 или 3): 2
Введите матрицу коэффициентов (2x2) построчно:
5 2
2 1
Введите вектор правых частей (2 значения):
7 9

Матрица системы:
    5.00     2.00 |     7.00
    2.00     1.00 |     9.00

Решение (метод Крамера):
x[1] = -11.00
x[2] = 31.00
```
