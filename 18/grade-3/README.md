# Оценка 3 — Базовые функции рисования

## Идея

Реализовать минимальный набор функций, достаточный для работы графического редактора: очистка холста, установка пикселя, кисть-круг и закрашенный прямоугольник.

## Что реализовано

Все функции находятся в `src/paint_core.cpp`:

| Функция | Что делает | Ключевой приём |
|---|---|---|
| `clearCanvas` | Заполняет весь холст белым цветом | `std::fill` по каждой строке |
| `drawPixel` | Ставит пиксель с проверкой границ | Условие `x ∈ [0, WIDTH)`, `y ∈ [0, HEIGHT)` |
| `drawBrush` | Рисует закрашенный круг заданного радиуса | Перебор `dx, dy`: `dx²+dy² ≤ r²` |
| `drawRectangle` | Рисует закрашенный прямоугольник по двум углам | `std::min`/`std::max` + двойной цикл |

## Запуск

```bash
mkdir build && cd build
cmake ..
make
./paint
```

## Ожидаемый результат

После запуска открывается окно 800×600. Доступны инструменты:
- **Кисть** (`B`) — рисует круглым пятном
- **Прямоугольник** (`R`) — рисует залитый прямоугольник перетаскиванием
- **Ластик** (`E`) — стирает белым
- **Очистка** (`C`) — сбрасывает холст в белый

> Инструменты Line (L), Undo (Ctrl+Z), фильтры (G/N/M) и выбор цвета по цифрам не реализованы в этой версии (оценка 4–5).
