# Оценка 4 — Комплексные числа

## Условие

Комплексное число `z = a + bj` представляется двумя отдельными переменными. Массив из N комплексных чисел хранится как два массива `re[]` и `im[]` размером N.

## Что делает программа

1. Считывает с клавиатуры размер N
2. Заполняет `re[]` и `im[]` случайными числами в диапазоне `[-32768..32767]`
3. Выводит сигналы в формате `s[i] = re + j*(im)`
4. Вычисляет энергию сигнала по формуле: `signal_energy = sum(re[k]² + im[k]²)`
5. Находит максимальный элемент среди реальной части

## Пример вывода

```
Введите количество комплексных чисел N: 3

Значения сигнала:
s[0] = -1234.0 + j*(-3000.0)
s[1] = 1000.0 + j*(1000.0)
s[2] = -3.0 + j*(-1.0)

Энергия сигнала: 3522010.0
Максимальный элемент реальной части: 1000.0
```
