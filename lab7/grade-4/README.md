# Вариант 3 — Оценка 4

## Задание

В заданном предложении найти слово, в котором количество буквы `'a'` максимально. Вывести найденное слово и посчитать, сколько раз буква `'a'` встречается в этом слове.

## Решение

Программа перебирает текст посимвольно, выделяя слова по пробелам. Для каждого слова считает количество букв `'a'` (учитывает как строчную, так и заглавную). Если текущее слово содержит больше букв `'a'`, чем предыдущий максимум — обновляет лучшее слово.

## Запуск

```bash
gcc zadanie1.c -o zadanie1
./zadanie1
```

## Пример вывода

```
aaaaaaaaaaaaaaaaaaaaaaaaaaLorem
28
```
